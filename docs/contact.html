联系我们

与 Facebook、Google 等不同，我们开放供所有人出于任何目的与我们联系。 您可以出于多种目的联系 One Future Unified Team，例如支持、反馈、错误报告、建议、捐赠、赞助等。请不要联系我们进行促销和垃圾邮件，我们不会向不相关的电子邮件发送任何回复 在我的邮箱里。

一个未来统一团队是一个忙碌的独立团队，我们团队的成员非常少，实际上整个团队只有 3 名成员，因此团队可能需要一段时间才能收到您的消息并回复您。 我们也将尽最大努力尽快回复每张票，请耐心等待。 我们也需要大约 3 个工作日才能回复。 我们也只在工作时间内通过官方平台回复，如下所示。

  • Facebook
    Facebook 是一个未来统一团队目前的主要社交媒体平台。 我们不时在我们的官方 Facebook 页面上发布公告、更新和新闻等内容。 您也可以通过 Facebook Messenger 直接与我们联系到我们的官方 Facebook 页面，以获得支持、错误报告、反馈和建议，并审查我们。
  • Twitter
    Twitter 是一个未来统一团队的第二个社交媒体平台。 我们也在我们的官方 Twitter 帐户中发布有关公告、更新和新闻的内容。 您也可以向我们发送消息请求，直接在 Twitter 上与我们联系以获得支持、错误报告、反馈和建议。
