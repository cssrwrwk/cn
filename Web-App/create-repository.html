Create Repository

We provide multiple methods for everyone to start and create their repository on Software and Website Developer Team. Users could choose one methods for all methods provided here to start and create a repository.

Methods

You could create a issue in our Documentation repository on Software and Website Developer Team Github organization. You could create an issue stating all information about your repository and etc so a volunteer could stand in and create the repository for you.

You could create a repository by creating a ticket to contact us. You could contact us and tell us information that you want in your repository so Software and Website Developer Team staff could create a repository for you. This will only work during Software and Website Developer Team staff business hours. You could create a ticket on our Discord/Support Desk.

That not all, you could also fill up Software and Website Developer Team repository Google Form. You could fill up the Google Form with all information in the fields needed. Once you finish filling up, submit the form to Software and Website Developer Team and wait for a repository creator or SWDT staff to create a repository for you, which usually took a few hours to days.
