Project Library

A collection of all open source projects. Contributing to open source projects can be intimidating. Especially when you're a beginner. That's why we've created this collection of open source projects that are beginner friendly and actively maintained by us. Take a look at the projects we have here, and we are sure you'll find something that might interests you.

The Project Library is a collection of beginner friendly open source projects. A place for beginner contributor to find projects to contribute to. The Project Library is a place for everyone to find active beginner friendly open soruce projects to check out and contribute.

The Project Library also an open source websites which is licensed under AGPL 3.0 for everyone to view and study our codes. You could view and contribute to our source code. You could help to contribute to the project on our Github repository.

The Project Library is currently powered by One Future Web Team, originated from a two person team, Jst Tan and Poetsec. Jst Tan is a web developer, story writer, video and wiki editor and Poetsec is a sysadmin whose main focus is on IT, shell scripting, hardware hacking, and the general security side of things.

The Project Library idea was born from a Reddit post in which someone mentioned how it can be difficult to find good (and actively maintained) projects to contribute to that are beginner friendly. We want to change that so the world of open source software and collaborating on the projects you love and use should be available and accessible to everyone.

Build your confidence

Coding can be hard and intimidating, but by contributing to open source projects you can build your confidence and expand your skillset. As you help with different projects, you have the opportunity to learn more than you might on your own.

Expand your portfolio

If you're new to the tech world, it can be hard to land a job without a strong portfolio. Working on open source projects shows passion and dedication and is attractive to potential employers, not to mention the personal satisfaction that comes with it.

Learn more with others

While internet articles and videos are a near infinate resource of learning material, you can learn even more from experienced developers who have been there and done that. Contributing to open source gives you the opportunity to connect and learn from these people.

Create with a community

We all use open source software in our daily lives, but have you ever wished a certain piece of software did something better? By contributing to these projects, you can be an influence in the community as you help to build the software you love and meet interesting people along the way too.
