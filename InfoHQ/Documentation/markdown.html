Markdown

InfoHQ currently also support markdown on all InfoHQ websites. InfoHQ users and admin could use markdown on every InfoHQ websites and pages. You could use any types of markdown available too.

Basic Style

InfoHQ offered many basic styles in markdown. You could use most if not all basic style markdown in InfoHQ.

  • Strong : < strong >Hi < / strong >
  • Italic : < em >Italic < / em >

Headings

InfoHQ also support heading markdown too. We support all 6 types of headings markdown in InfoHQ without any issues.

Results

  • H1

  • H2

  • H3

  • H4

  • H5
  • H6

Codes

  • < h1 >H1< / h1 >
  • < h2 >H2< / h2 >
  • < h3 >H3< / h3 >
  • < h4 >H4< / h4 >
  • < h5 >H5< / h5 >
  • < h6 >H6< / h6 >

Line Break

In order to use line break, you could use br when you wanted to make the linebreak. We currently support br markdown in InfoHQ. You could use br in any place that you wish to use line break.

Codes Block

InfoHQ also support code block markdown too! You could denote a word or phrase as code using code block by putting code in front and behind the codes
