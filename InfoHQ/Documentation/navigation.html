Navigation

Each InfoHQ come with an installed navigation for InfoHQ founders and users. All InfoHQ users and admin could changed the navigation anytime when needed.

In order to change site navigation, please check all pages in your InfoHQ. Most InfoHQ pages will have a prebuilt UI made by Software and Website Developer Team. In 'body', you will see a id named 'overlay-content', you could then changed everything inside 'overlay-content'. Once you done, make a pull request and once it approved, you will be able to check out your new site navigation.

Remember to changed it on all pages in your InfoHQ except redirects pages.
