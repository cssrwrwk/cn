Installing

InfoHQ is a free open source software. All users could view, study and read all of InfoHQ source codes. All users could download and install InfoHQ on their websites for free per our licenses terms (MIT open source licenses).

Users could install InfoHQ software on their sites too. Installing InfoHQ is totally free for everyone in our sides. You still needed web hosting to install InfoHQ in through and if you want, a domain would be needed. We provide and linked some free hosting example here:

  • Github Pages (Recommended)
  • Vercel
  • Netlify
  • Cloudflare

Installing

In order to install InfoHQ, you firstly needed a web hosting that support hosting for static sites. Mostly free and paid web hosting provide support for static sites too. You could download InfoHQ in Github repositoey release tab. You could download any version you likes too here. After downloading, please upload to your hosting file section. Once you uploaded and created the file, edit the "Edit", "View Source" and "History" links and push it again. After the push succeed, your InfoHQ is now up and running for you to create and edit articles.
