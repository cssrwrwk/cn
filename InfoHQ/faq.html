InfoHQ is an open source software, based on HTML, CSS and JavaScript. Users could edit InfoHQ pages and view InfoHQ pages history and source codes.

InfoHQ got some inspiration from "wiki" by MediaWiki. InfoHQ is currently available to be hosted in most (if not all) free and paid web hosting companies.

One Future Unified Team is a small team based in Malaysia. One Future Unified Team is focused in many things, from web development to story writing. One Future Unified Team is founded from a group of young friends with an interest in technology.

InfoHQ is an open source software. All users could view and edit all of our source code. InfoHQ is licensed under MIT open source licenses.

InfoHQ is a free open source software for everyone! Everyone could use and edit InfoHQ for free. All features provided in InfoHQ is totally free for everyone to use and enjoy.

Everyone could create and request a InfoHQ in One Future Unified Team. Creating a InfoHQ is totally free in One Future Unified Team. Please fill up the Google Form linked and provided here to request an InfoHQ.
